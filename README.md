# The Cartographer - Fork for Linux Kernel 6.5.5

Tested on Arch, built with:
*The following Kernel Build configurations
	*CONFIG_FTRACE
	*CONFIG_KALLSYMS
	*CONFIG_DYNAMIC_FTRACE_WITH_REGS
	*CONFIG_HAVE_FENTRY
	*CONFIG_LIVEPATCH LIVE
